## Features

- Responsive navigation bar
- Reusable layout component
- Home page with hero section
- About page with skills and timeline
- Contact page with contact form
- Custom 404 page
- React Router navigation
- Pure CSS styling
- Mobile-friendly design

- CSS3

- Vite
